# 🇵🇭 Software Engineering Roadmap 2026
### For Fresh Graduates in the Philippines

> **Your launchpad from diploma to developer — built for the Philippine tech landscape.**

---

## 📍 Overview

Breaking into the software engineering industry as a fresh graduate in the Philippines is absolutely achievable — the local and remote job markets are growing fast. This roadmap gives you a realistic, month-by-month guide tailored to PH opportunities, salaries, and culture.

---

## 🎓 Phase 0 — Before You Graduate (Final Semester)

**Goal:** Don't wait for graduation. Start now.

### ✅ Checklist
- [ ] Finalize your **capstone/thesis** — make it deployable and put it on GitHub
- [ ] Build or clean up your **GitHub profile** (recruiters will look at this)
- [ ] Set up your **LinkedIn** — use your university email while you still can
- [ ] Attend at least one **DevCon** or local tech event before graduation

### 💡 Tip
Your **capstone project is your first portfolio piece**. Make sure it's live, documented, and shareable.

## 🗓️ Month 1–2 — Foundation Lock-In

**Goal:** Solidify your core skills and pick your specialization track.

### 🛤️ Pick Your Track

| Track | Best For |
|---|---|
| **Frontend** | Creative, UI-focused |
| **Backend** | Logic, APIs, databases |
| **Full Stack** | Versatility, startups |
| **Mobile (Android/iOS/Flutter)** | Apps, freelance |
| **QA/Automation** | Testing, detail-oriented |
| **DevOps/Cloud** | Infrastructure, remote jobs |

### 🔧 Universal Foundations (All Tracks)
- **Version Control:** Git & GitHub — branches, PRs, commit messages
- **Command Line:** Basic Linux/Bash commands
- **Algorithms & Data Structures:** Practice on [LeetCode](https://leetcode.com) or [HackerRank](https://hackerrank.com) — at least Easy level
- **Networking basics:** HTTP, REST, DNS (watch Fireship or Traversy Media on YouTube)
- **English communication:** Most PH tech companies and remote clients use English

### 📚 Free Learning Resources (PH-Friendly)
- [freeCodeCamp](https://freecodecamp.org) — free certifications
- [The Odin Project](https://theodinproject.com) — full stack curriculum
- [CS50 by Harvard](https://cs50.harvard.edu) — free, respected globally
- YouTube: Traversy Media, Fireship, Kevin Powell (CSS)
- [Udemy](https://udemy.com) — wait for sales (courses drop to ₱450–₱650)

## 🗓️ Month 3–4 — Build Your Portfolio

**Goal:** Have 3 real projects that show you can build, not just learn.

### 🏗️ Project Ideas by Track

**Frontend:**
- Personal portfolio website (must-have)
- Weather app using OpenWeatherMap API
- E-commerce UI with cart functionality

**Backend:**
- REST API for a simple inventory system
- Authentication system (JWT, OAuth)
- CRUD app with PostgreSQL or MySQL

**Full Stack:**
- Job board app (CRUD + auth + search)
- Blog platform with CMS
- Real-time chat using WebSockets

**Mobile:**
- Expense tracker app
- Notes app with local storage
- Weather or news app with public API

### 🌟 Portfolio Tips for PH Market
- Host everything — use **Vercel**, **Netlify**, or **Railway** (free tiers)
- Write a `README.md` for every project — explain what it does, tech used, how to run it
- Show **problem-solving**, not just UI — include a "Challenges" section in your README
- Bonus: Write a short **dev.to** or **hashnode** blog about what you built

## 🗓️ Month 5–6 — Job Hunt Begins

**Goal:** Start applying, even if you feel "not ready." You never will be — start anyway.

### 📋 Where to Find PH Tech Jobs

**Local (On-site / Hybrid):**
- [JobStreet Philippines](https://jobstreet.com.ph)
- [LinkedIn Jobs PH](https://linkedin.com/jobs)
- [Kalibrr](https://kalibrr.com)
- [OnlineJobs.ph](https://onlinejobs.ph)
- Company career pages: Accenture PH, Exist Software, QBO (startup hub), Globe Fintech, Maya, GCash

**Remote / International Clients:**
- [Upwork](https://upwork.com) — freelance, great for building income while job hunting
- [Toptal](https://toptal.com) — rigorous, higher pay
- [Remote OK](https://remoteok.com)
- [We Work Remotely](https://weworkremotely.com)
- [Deel Jobs](https://deel.com/jobs)

### 📄 Resume Tips
- Keep it **1 page**, clean, ATS-friendly format
- List: Projects > Skills > Education (for fresh grads, projects come first)
- Tailor keywords to match job descriptions
- Use [Overleaf](https://overleaf.com) (LaTeX templates) or [Reactive Resume](https://rxresu.me) (free)
- Include: GitHub link, LinkedIn, portfolio URL, location (mention "open to remote")

### 💬 Common PH Tech Interview Stages
1. **HR Screen** — background, salary expectations, availability
2. **Technical Exam** — online coding test (HackerRank, Codility, or take-home)
3. **Technical Interview** — live coding or whiteboard, CS concepts, past projects
4. **Final Interview** — culture fit, behavioral (STAR method)

## 🗓️ Month 7–9 — First Job & Leveling Up

**Goal:** Land your first role and survive the first 90 days.

### 🧭 Your First 90 Days
- **Day 1–30:** Observe, ask questions, understand the codebase — don't try to be a hero
- **Day 31–60:** Take ownership of small tasks, ship something real
- **Day 61–90:** Build relationships with your team, understand the product

### 🛠️ Skills to Build On the Job
- Code review culture — learn to give and receive feedback
- Agile/Scrum workflow — sprints, standups, retrospectives (used in most PH IT companies)
- Jira, Confluence, or Notion — project management tools
- Proper Git workflows (branching strategies, PRs, merge conflicts)

### 📈 How to Stand Out Early
- Document what you learn in a private Notion or Obsidian vault
- Ask your TL/manager: "What does success look like for me in 6 months?"
- Volunteer for code reviews — it forces you to read other people's code
- Attend internal tech talks or company lunch-and-learns

## 💡 Filipino Dev Reality Check

### The BPO Trap
Many PH IT companies have BPO-adjacent culture — repetitive work, low innovation. Aim for:
- **Product companies** (they build their own software)
- **Startups** (steep learning curve, more ownership)
- **Remote-first companies** (pay in foreign currency)

### The Remote Work Goal
Remote work paying USD rates is the **highest-leverage career move** for Filipino devs. A $3,000/month USD salary equals ~₱170,000+ — life-changing in the PH cost of living. This is achievable within 2–3 years.

### Taxation & Freelance Setup
- If freelancing: Register with **BIR** as a self-employed individual (Form 1901)
- Consider opening a **USD bank account**: UnionBank, BDO, or a digital bank like Maya (for forex rates)
- Learn about **TRAIN Law** tax brackets for income planning
